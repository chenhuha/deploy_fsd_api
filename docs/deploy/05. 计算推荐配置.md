# 计算推荐配置通用

**接口地址** `/api/deploy/node/reckRecommendConfigCommon`

**请求方式** `POST`

**consumes** `["application/json"]`

**请求示例**

```json
{
    "cephCopyNumDefault": 2,
    "cephServiceFlag": true,
    "localServiceFlag": true,
    "nodes": [
        {
            "nodeIP": "172.27.134.201",
            "nodeType": [
                "COMPUTE",
                "CONTROL",
                "STORAGE"
            ]
        }
    ],
    "serviceType": [
        "VOI",
        "VDI"
    ],
    "storages": [
        {
            "name": "sda",
            "cache2data": [],
            "purpose": "SYSTEM",
            "size": "558.91 GB"
        },
        {
            "name": "sdb",
            "cache2data": [],
            "purpose": "CEPH_DATA",
            "size": "558.91 GB"
        },
        {
            "name": "sdc",
            "cache2data": [],
            "purpose": "CEPH_DATA",
            "size": "558.91 GB"
        },
        {
            "name": "sdd",
            "cache2data": [],
            "purpose": "LOCAL_DATA",
            "size": "558.91 GB"
        },
        {
            "name": "nvme0n1",
            "cache2data": [
                "sdb"
            ],
            "purpose": "CEPH_CACHE",
            "size": "1.09 TB"
        }
    ]
}
```

### 请求参数

NodeCommonReckonRecommendConfig

| 参数名称             | 参数说明             | 请求类型 | 是否必须 | 数据类型       | schema               |
| -------------------- | -------------------- | -------- | -------- | -------------- | -------------------- |
| cephCopyNumDefault   | Ceph 副本数          | body     | false    | integer(int32) |                      |
| [cephServiceFlag]()  | ceph 服务是否开启    | body     | true     | boolean        |                      |
| [localServiceFlag]() | 本地存储服务是否开启 | body     | true     | boolean        |                      |
| nodes                | 节点信息集合         | body     | true     | array          | DeployNodeCheckDTO   |
| serviceType          | 服务类型             | body     | true     | array          |                      |
| storages             | 存储信息             | body     | true     | array          | DeployNodeStorageDTO |

NodesDTO

| 参数名称 | 参数说明 | 请求类型 | 是否必须 | 数据类型 | schema |
| -------- | -------- | -------- | -------- | -------- | ------ |
| nodeIP   | 节点 IP  | body     | true     | string   |        |
| nodeType | 节点类型 | body     | false    | array    |        |

StorageDTO

| 参数名称    | 参数说明                                                  | 请求类型 | 是否必须 | 数据类型 | schema |
| ----------- | --------------------------------------------------------- | -------- | -------- | -------- | ------ |
| cache2data  | 如果是 cache 盘管理的 data 盘集合                         | body     | false    | array    |        |
| name        | 磁盘名称                                                  | body     | true     | string   |        |
| [purpose]() | 磁盘用途,可用值:SYSTEM、CEPH_DATA、CEPH_CACHE、LOCAL_DATA | body     | true     | string   |        |
| size        | 大小                                                      | body     | true     | string   |        |


### 响应参数

NodeRecommendConfigVO

| 参数名称         | 参数说明              | 类型   | schema |
| ---------------- | --------------------- | ------ | ------ |
| commonCustomCeph | 部署通用自定义 CephVO | int    |        |
| commonCustomPool | 部署通用自定义 PoolVO | int    |        |
| [cephSizeMax]()  | ceph的可用的最大值    | string |        |
| [localSizeMax]() | local 存储最大值      | string |        |

CommonCustomCephVO

| 参数名称           | 参数说明        | 类型           | schema |
| ------------------ | --------------- | -------------- | ------ |
| cephCopyNumDefault | Ceph 默认副本数 | integer(int32) |        |

CommonCustomPoolVO

| 参数名称         | 参数说明         | 类型           | schema |
| ---------------- | ---------------- | -------------- | ------ |
| cephfsPoolPgNum  | cephfs 池 pg 数  | integer(int32) |        |
| cephfsPoolPgpNum | cephfs 池 pgp 数 | integer(int32) |        |
| imagePoolPgNum   | 镜像池 pg 数     | integer(int32) |        |
| imagePoolPgpNum  | 镜像池 pgp 数    | integer(int32) |        |
| volumePoolPgNum  | 卷池 pg 数       | integer(int32) |        |
| volumePoolPgpNum | 卷池 pgp 数      | integer(int32) |        |

### 响应示例

```json
{
    "code": 0,
    "data": {
        "cephSizeMax": "447.13GB",
        "commonCustomCeph": {
            "cephCopyNumDefault": 2
        },
        "commonCustomPool": {
            "cephfsPoolPgNum": 32,
            "cephfsPoolPgpNum": 32,
            "imagePoolPgNum": 8,
            "imagePoolPgpNum": 8,
            "volumePoolPgNum": 32,
            "volumePoolPgpNum": 32
        },
        "localSizeMax": "558.91GB"
    },
    "message": null,
    "path": null,
    "state": 200,
    "timestamp": 1686799106994
}
```

# 计算推荐配置个性化

**接口地址** `/api/deploy/node/showRecommendConfig`

**请求方式** `POST`

**consumes** `["application/json"]`

### 请求示例

```json
{
    "cephCopyNumDefault": 2,
    "cephServiceFlag": true,
    "localServiceFlag": true,
    "nodes": [
        {
            "nodeIP": "172.27.134.201",
            "storages": [
                {
                    "name": "sda",
                    "purpose": "SYSTEM",
                    "size": "558.91 GB"
                },
                {
                    "name": "sdb",
                    "purpose": "CEPH_DATA",
                    "size": "558.91 GB"
                },
                {
                    "name": "sdc",
                    "purpose": "CEPH_DATA",
                    "size": "558.91 GB"
                },
                {
                    "name": "sdd",
                    "purpose": "LOCAL_DATA",
                    "size": "558.91 GB"
                },
                {
                    "name": "nvme0n1",
                    "purpose": "CEPH_CACHE",
                    "size": "1.09 TB",
                    "cache2data": [
                        "sdb",
                        "sdc"
                    ],
                    "hddName": [
                        {
                            "name": "sdb",
                            "disabled": false
                        },
                        {
                            "name": "sdc",
                            "disabled": false
                        },
                        {
                            "name": "sdd",
                            "disabled": false
                        }
                    ]
                }
            ]
        }
    ],
    "serviceType": [
        "VOI",
        "VDI"
    ]
}
```

### 响应示例

```json
{
    "code": 0,
    "data": {
        "cephSizeMax": "447.13GB",
        "commonCustomCeph": {
            "cephCopyNumDefault": 2
        },
        "commonCustomPool": {
            "cephfsPoolPgNum": 32,
            "cephfsPoolPgpNum": 32,
            "imagePoolPgNum": 8,
            "imagePoolPgpNum": 8,
            "volumePoolPgNum": 32,
            "volumePoolPgpNum": 32
        },
        "localSizeMax": [
            "558.91GB"
        ]
    },
    "message": null,
    "path": null,
    "state": 200,
    "timestamp": 1686812239925
}
```

> [请求参数与响应参数同上。
> 注意 `计算推荐配置通用` 时，`localSizeMax` 返回的是一个字符串; `计算推荐配置个性化` 时，`localSizeMax` 返回的是一个列表。]()
